# Project Green River — A Modern Azure Reconstruction

**How modern data analytics could save lives in cold-case investigations.**

An interactive, single-file case study that rebuilds the Green River Killer
investigation as a modern Microsoft Azure data platform — mapped to the
**DP-900 (Azure Data Fundamentals)** exam.

It walks every era-appropriate evidence type (handwritten field notes, witness
statements, the 10,000+ public tips, vehicle/motel/employment records,
dump-site geography, and forensic samples) through the full data lifecycle:
**ingest → store → process → analyze → visualize** — and argues how a queryable
platform could have surfaced Ridgway years before the 2001 DNA match.

## View it
Open `index.html` in any browser. No build step, no dependencies — everything
(including all graphics) is inline.

## Publish with GitHub Pages
1. Push these files to a GitHub repo.
2. Repo **Settings → Pages → Build and deployment**.
3. Source: **Deploy from a branch**, Branch: **main**, Folder: **/ (root)**.
4. Save. Your site appears at `https://<username>.github.io/<repo>/` in ~1 min.

## Note
Educational thought experiment about a real tragedy. Case facts are cited in the
page footer. All visuals are original illustrations; no victim or news
photographs are used.
